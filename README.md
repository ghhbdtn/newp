# newp
## Node.js - v16.17.0
## npm - 8.15.0 (version)
## Установка зависимостей
```
npm install
```
 
### Компиляция (+hot reload) проекта для разработки
```
npm run serve
```

### Сборка проекта
```
npm run build
```

## Разворачивание проекта в docker:
### docker-compose.yaml файл лежит в корневой папке проекта, для запуска всех контейнеров необходимо: 
```
docker-compose up
```
